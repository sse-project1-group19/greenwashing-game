# greenwashing-game
This repo hosts the code for our "Greenwashing Game" site. Undertaken as our Project 2 submission for CS4575 "Sustainable Software Engineering" at TU Delft, NL

## Development
1. Clone the repo
2. Use ``npm install`` to install dependencies
3. Run ``npm start`` to run the dev server
4. The app will be available at `http://localhost:3000`

## Deployment
1. Build the app for production with `npm run build`
2. Deploy to GitHub Pages with `npm run deploy`
3. The website will be live at `https:///sse-project1-group19.github.io/greenwashing-game`
